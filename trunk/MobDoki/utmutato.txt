MobDoki - telepítési útmutató

1. Adatbázis:
	- postgresql
	- Adatbázis tábláinak létrehozása:
		.\postgresql\MobDoki DB create.sql

	- Adatbázis feltöltése adatokkkal:
		.\postgresql\MobDoki DB fill.sql
	- Utólagos módosítások az adatbázison:
		MobDoki DB table alter.sql


2. Webszerver:
	- Netbeans 7.0 + Apache Tomcat 7.0.11


3. Android kliens:
	- Webserver IP címének beállítása
		.\mobdoki\src\mobdoki\client\connection\Connection.java
		
	- A Google Maps mûködéséhez, az API Key beállítása
		.\mobdoki\res\values\apikey.xml
		
	!!! Ezt a két fájlt SVN-en nem kell frissíteni, mert úgyis minden gépen más!

	Az API Key-hez egy kis segítség:
	MD5 fingerprint generálás MapView használatához: 
	1.  Kell lennie egy debug.keystore nevû fájlnak a C:\Users\<felhasználóneved>\.android könyvtárban. Ezt másold át a C:\Java\jdk<verziószám>\bin könyvtárba 
	2.  commans prompt-ban navigálj a Java\jdk<verziószám>\bin könytvárba, és írd be a következõ parancsot: keytool.exe –list –alias androiddebugkey –keystore debug.keystore –storepass android –keypass android 
	3.  ez elvileg kiadja az MD5 fingerprintet 
	4.  itt pedig ebbõl lehet generálni az API Key-t: http://code.google.com/intl/hu/android/maps-api-signup.html
